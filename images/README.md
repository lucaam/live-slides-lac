# Cartella Immagini

Questa cartella contiene le immagini per la presentazione.

## Immagini usate (da Unsplash)

Le slide ora usano immagini da Unsplash tramite URL per:
- Pizza Napoletana
- Farina Tipo 00
- Pomodori San Marzano
- Mozzarella di Bufala
- Pizza nel forno

## Per usare immagini locali

1. Scarica le tue foto e mettile in questa cartella
2. Nel file `index.html`, sostituisci gli URL di Unsplash con:
   ```html
   src="images/tua-foto.jpg"
   ```

## Dimensioni consigliate

- Slide full-width: 1920x1080px
- Grid items: 600x400px
- Backgrounds: 1920x1080px o superiore

## Formati supportati

- JPG/JPEG (consigliato per foto)
- PNG (per immagini con trasparenza)
- WebP (per dimensioni ridotte)
